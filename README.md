## 🚀 快速开始

✅ 已验证兼容版本：
- Python 3.9.13 
- Node.js v16.14.2+
- npm 8.5.0+

ℹ️ 更低/更高版本可能正常工作，但未经全面测试


## 🛠 安装指南

### 1. 克隆仓库
```bash
git clone https://github.com/Julie3399/stock_monitor.git
cd stock-monitor
```
### 2. 初始化环境
```bash
bash setup.sh
```
📌 脚本将自动完成：
- Python依赖安装
- Node模块安装

## 🔥 使用指南
启动系统
```bash
python run.py
```
📌 脚本将自动完成：
- 启动后端服务
- 启动前端服务
- 自动打开浏览器并显示平台界面
